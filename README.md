# MinimizeYTM

YouTube Music をタスクトレイから操作できるシンプルな常駐アプリケーションです。

## 概要

MinimizeYTM は YouTube Music のウィンドウをタスクトレイから表示/非表示に切り替えられるツールです。Rust で開発されており、軽量で高速に動作します。

## 機能

- **タスクトレイ常駐**: アプリケーション実行中、タスクトレイに常駐します
- **カスタムアイコン表示**: タスクトレイにカスタムアイコンを表示
- **ダブルクリックでトグル**: トレイアイコンをダブルクリックすると YouTube Music のウィンドウを表示/非表示に切り替えます
- **Win + Y ホットキー**: `Win + Y` でも YouTube Music のウィンドウを表示/非表示に切り替えます
- **右クリックメニュー**: トレイアイコンを右クリックすると終了メニューが表示されます
- **終了時の自動復元**: アプリケーションを終了すると、非表示にしていたウィンドウを自動的に表示します

## 使い方

### 実行方法

```bash
cargo run
```

または、リリースビルドを実行：

```bash
cargo run --release
```

実行すると以下の動作が行われます：

1. アプリケーションがタスクトレイに常駐します
2. トレイアイコンをダブルクリックすると YouTube Music のウィンドウが表示/非表示に切り替わります
3. `Win + Y` を押しても YouTube Music のウィンドウが表示/非表示に切り替わります
4. トレイアイコンを右クリックして「終了」を選ぶとアプリケーションが終了します

実行ファイルは `target/release/MinimizeYTM.exe` から直接実行できます。パスを通しておくと、どのディレクトリからでも `MinimizeYTM.exe` で実行できて便利です。

## トラブルシューティング

### YouTube Music のウィンドウが見つからない

アプリケーション実行時に「YouTube Music のウィンドウが見つかりませんでした」というメッセージが表示される場合、YouTube Music が起動していることを確認してください。

### トレイアイコンが表示されない

Windows のタスクトレイ設定で、通知領域に表示するアプリケーションを確認してください。隠れたアイコンの中に表示されている可能性があります。

### Win + Y が反応しない

`Win` キーとの組み合わせは Windows 側の予約ショートカットと競合する場合があります。反応しない場合は、トレイアイコンのダブルクリック操作を利用してください。

## 開発者向け

### プロジェクト構成

- `src/main.rs` - メインエントリーポイント
- `src/tray.rs` - タスクトレイ操作関連（トレイアイコン、メッセージループ、イベント処理）
- `src/window.rs` - ウィンドウ操作関連（ウィンドウ検索、表示/非表示制御）
- `src/icon.rs` - アイコンローダー（ICOファイルの読み込みと変換）
- `assets/icon.ico` - タスクトレイに表示されるカスタムアイコン
